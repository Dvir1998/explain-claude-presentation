💬 עבודה עם שיחות

פתיחת שיחה חדשה

# בתיקייה שלך
claude

# עם שאלה ראשונית
claude "הסבר לי את הקוד הזה"

המשך שיחה אחרונה

claude --continue
# או בקיצור:
claude -c

בחירת שיחה מהיסטוריה

claude --resume
# או בקיצור:
claude -r

# לשיחה ספציפית
claude -r "שם-השיחה"
💡 עבודה עם כמה שיחות במקביל
  1. פתח טאב חדש: Cmd+T (Mac)
  2. עבור לתיקייה אחרת: cd /other/project
  3. פתח Claude: claude
עכשיו יש לך שתי שיחות נפרדות!

🖥️ אפשרויות שורת פקודה

הכי חשובות

פקודה מה עושה
claude שיחה חדשה
claude -c המשך שיחה אחרונה
claude -r בחר מהיסטוריה
claude -p "שאלה" שאל בלי אינטראקציה
claude --model opus השתמש במודל Opus
claude --model haiku השתמש במודל Haiku

דוגמאות שימוש

# שאל שאלה מהירה
claude -p "מה הגרסה של Node.js?"

# המשך עם מודל חזק יותר
claude --continue --model opus

# עבוד במצב תכנון
claude --permission-mode plan

💡 טיפים חשובים
